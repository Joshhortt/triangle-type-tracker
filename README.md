# TRIANGLE TYPE TRACKER


## Introduction

_Keep practicing **jquery** - **Multiple Branching**: (if', else if', else', return') statements_.<br/>
_This web app returns triangle types and tracks dimensions_,<br/>
_I used very often the **===** (triple equals) and the **||** (OR) operators together with: .show()_;<br/>
_.hide(); .val() and .submit(); .parseFloat(); .isNaN() and .append() functions/methods_.<br/>
_version: 26/04/2019_<br/>
_By **Josh Hortt**_

## Description

_The Web-page asks the user to enter some numbers (length-size) in three different inputs_.<br/>
_After that, the User clicks the button **Check**, that shows the User_.<br/>
_what type of triangle it is. If the app accepts these numbers_<br/>
_it will be send to one of the columns (Triangle Types), on the right_.<br/>
_The User will be alerted with three different **ERROR** messages if certain_<br/>
_requirements are not valid according to javascript code_.
